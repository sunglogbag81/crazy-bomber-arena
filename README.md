# Crazy Bomber Arena

크레이지 아케이드에서 영감을 받은 발표용 실시간 온라인 폭탄 대전 웹게임입니다.

## 기능

- 방 코드 기반 온라인 대전
- Socket.IO 실시간 이동/폭탄 동기화
- 2~4인 플레이
- 파괴 가능한 블록, 폭탄 연쇄 폭발, 체력/승패 처리
- 키보드와 모바일 터치 조작 지원

## 실행

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속 후 방을 만들고, 다른 브라우저/기기에서 방 코드로 입장하세요.

## 조작

- 이동: WASD 또는 방향키
- 폭탄: Space
- 모바일: 화면 버튼

## 발표 포인트

1. 서버가 방 상태를 authoritative하게 관리합니다.
2. 클라이언트는 입력만 보내고, 서버가 매 tick마다 보드/플레이어/폭탄 상태를 브로드캐스트합니다.
3. 방 코드만 공유하면 같은 네트워크/배포 URL에서 바로 대전할 수 있습니다.

## 배포

Node.js 실행이 가능한 Render, Railway, Fly.io 같은 서비스에 올리면 온라인으로 플레이할 수 있습니다.
`PORT` 환경변수는 자동 인식합니다.
