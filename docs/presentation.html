Part 7 / How to Join

서비스 접속

같은 네트워크에서는 내부 IP로 접속하고, 외부 발표용은 공유기 포트포워딩 또는 도메인 연결을 사용합니다.

Local

같은 와이파이

http://192.168.0.4:3000
3000번 포트가 게임 서버
80번은 AdGuardHome이 사용 중
Online

외부 접속

http://도메인:3000
외부 3000 → 192.168.0.4:3000 TCP 포트포워딩
방 코드 공유 후 친구 입장
